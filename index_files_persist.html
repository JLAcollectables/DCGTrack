Help & Import/Export

Export / Import Progress

  • Export: Downloads a JSON snapshot of all owned checkboxes (<Set>::<CardNumber>).
  • Import: Load a previously exported JSON to restore your progress.

Add/Paste CSV (single set)

  1. Paste CSV with columns exactly: Card Number, Name, Level (or -), Colour, Type, Rarity.
  2. Enter the set name (e.g., BT23).
  3. Click “Add Set”. Your data will appear and be saved locally.

Note: This build embeds your provided XLSX data for offline use. For bulk CSV import later, you can select multiple CSV files instead of a ZIP using your browser’s file picker in a future update.
